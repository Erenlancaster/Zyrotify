# Zyrotify
Zyrotify, multi spotify program checker,creator,follower etc.

**Features of script**
 - Clean menu
 - Spotify account checker (fix)
 - Spotify user/artist follower/unfollower
 - Spotify playlist follower/unfollower
 - Spotify account creator
